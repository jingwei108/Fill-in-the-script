🚀 方式一：一键安装脚本（自动批量）

安装后进入量化评教页面，点一下按钮即可自动完成所有课程。

  2. 点击下方按钮安装脚本
  3. 登录教务系统，进入“量化评教”页面
  4. 点击右上角橙色“一键自动评教”按钮

❓ 常见问题

脚本会不会盗取我的账号？

不会。脚本只在你的浏览器本地运行，不会向任何服务器发送账号密码。

安装脚本后看不到按钮？

请确认 Tampermonkey 扩展已启用，并且刷新教务系统页面。部分浏览器需要允许扩展在 jw.hstc.edu.cn 上运行。如果仍然不行，请使用“书签小工具”。

可以修改哪一题填“良”吗？

可以。安装 Tampermonkey 脚本后，在编辑器里找到 liangIndex，0 表示第一题，-1 表示最后一题，null 表示随机。
